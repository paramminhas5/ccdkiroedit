# Disco audio loop

Drop your **120 BPM ambient loop** here as `disco-loop.mp3`.

The file is wired up automatically — when the user clicks the 🪩 disco
toggle in the nav, it fades in. The 🔊/🔇 button next to the toggle
mutes it. If the file is missing, audio is silently skipped (no errors).

Recommended:
- Format: MP3 (best browser support) or OGG
- Duration: 8–32 seconds (it loops)
- Loudness: -16 to -20 LUFS so it sits under the UI
